## Question 1: q1 [2]

What is $\frac{d}{dx}(x^3)$?

Consider the **power rule**: $\frac{d}{dx}(x^n) = nx^{n-1}$

- ( ) $2x$
- (x) $3x^2$
- ( ) $x^3$
- ( ) $\frac{1}{x}$

> Apply the power rule with $n = 3$: bring down the exponent as a coefficient and reduce the power by one.

## Question 2: q2 [3]

Which of the following are trigonometric functions?

- [ ] $\ln(x)$
- [x] $\sin(x)$
- [x] $\cos(x)$
- [ ] $e^x$

> $\sin$ and $\cos$ are the fundamental trigonometric functions. $\ln(x)$ is logarithmic and $e^x$ is exponential.

## Question 3: q3 [5]

What is $\frac{d}{dx}(\sin x)$ evaluated at $x = 0$?

= 1

> $\frac{d}{dx}(\sin x) = \cos x$, and $\cos(0) = 1$.

## Question 4: q4 [2]

The derivative of a constant function $f(x) = 7$ is:

- (x) $0$
- ( ) $7$
- ( ) $1$
- ( ) Undefined

> The derivative of any constant is zero, since constants do not change.

## Question 5: q5 [3]

Select all functions whose derivative is $2x$:

- [x] $x^2$
- [x] $x^2 + 5$
- [ ] $2x^2$
- [ ] $x^3$

> Both $x^2$ and $x^2 + 5$ have derivative $2x$. The constant $5$ vanishes under differentiation.

## Question 6: q6 [5]

Compute $\frac{d}{dx}(3x^4)$ at $x = 1$.

= 12

> $\frac{d}{dx}(3x^4) = 12x^3$. At $x = 1$: $12 \cdot 1^3 = 12$.
